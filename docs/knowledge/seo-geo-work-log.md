### [2026-06-19] 사이트맵 생성 및 신선도 우선순위 설정 (A · 1차 구축)

기법: SEO · GEO
상태: 검증 대기

**why**
검색 유입 증대가 사이트 핵심 목표. 건강 칼럼 1,042편 등 총 1,053개 URL 보유. 검색엔진이 전체 페이지를 수집하려면 URL 목록 명시 필요. 그러나 사이트맵 부재 상태였음. 콘텐츠 존재해도 미색인 시 검색 노출 없음.

**before**
사이트맵 없음. /sitemap.xml 요청 시 404 반환.

**after**
사이트맵에서 총 1,053개 URL 제공(정적 5·진료 6·포스트 1,042). 각 URL에 최종수정일·갱신 빈도·우선순위 포함. 포스트는 발행 경과일 기준 6단계 우선순위(1일 이내 1.0 ~ 180일 이상 0.5). 서치콘솔 제출, 1,053 URL 처리 확인.

**how**
전체 포스트·진료 페이지 자동 수집, 정적 페이지 수동 등록. 발행일 기준 우선순위 부여 후 서치콘솔 제출.

**기대효과 / 측정 지표**
[예상] 미색인 글까지 색인되어 색인률 상승.
[측정 지표] 서치콘솔 > 색인 생성 > 페이지 (색인됨 수 / 1,053 대비)

**before**

**after**
전체 1,042편에 글 단위 구조화 데이터 동적 적용. 검색 결과 날짜 표시 확인(예: 2026. 5. 17.). 포함 항목: 제목·요약문·대표 이미지·발행일·수정일·글 주소·담당 저자·발행 기관·주제 분류·무료 열람 표기·언어(한국어).

**how**
포스트 데이터에서 발행일 추출해 표준 형식으로 변환. 구조화 데이터 동적 생성, 전체 포스트 자동 적용.

**기대효과 / 측정 지표**
